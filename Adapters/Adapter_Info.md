Adapters are python scrypts used to bump from a older version to a newer one.
Adapters are downloaded INSTEAD of direct update from 1.0.0.X0
Adapters are saved in TEMP folder for the update time. They contain the whole update proces, doing modular update, and bumping from a version to another systematicaly. (0.9.8.X3 -> 0.9.9.X0 -> 0.9.9.X1)
(that was the bump scheme from 0.9.8.X3 to 0.9.9.X1)
Adapter are very good, since each update does not modufy a stationary file, but a flash file. This means that it does not need updates trough himself.
