# Code of Conduct

## Our Pledge

V0RTEX is a malware analysis tool built for the security community. Everyone who participates — whether contributing code, opening issues, or joining discussions — is expected to maintain a respectful and professional environment.

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, background, experience level, or any other personal characteristic.

## Our Standards

**Expected behavior:**

- Be respectful and constructive in all interactions
- Provide helpful, specific feedback when reviewing contributions
- Accept feedback gracefully
- Focus discussions on technical merit

**Unacceptable behavior:**

- Harassment, insults, or personal attacks of any kind
- Sharing malware samples, live payloads, or weaponized code in issues or PRs
- Using V0RTEX or its codebase to facilitate actual attacks or harm
- Doxxing or sharing private information without consent

## Scope

This Code of Conduct applies to all project spaces: GitHub Issues, Pull Requests, Discussions, and any other official channels.

## Enforcement

Violations can be reported by opening a private issue or contacting the maintainer directly via GitHub. All reports will be handled confidentially.
